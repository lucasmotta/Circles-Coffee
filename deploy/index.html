• Coffee Circles

    My first experiment using css/html/js to create some kind of interaction using DOM objects (not canvas or webgl) and also to run on desktops and touchscreen devices.

    Every circle you see on the right side is a list item and you can drag them and play around.

    You can get the source at github.

About

This experiment was created by Lucas Motta and it was developed using Sublime Text 2, CoffeeScript, jQuery and TweenMax.
